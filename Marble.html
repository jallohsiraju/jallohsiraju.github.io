Marble Solitaire Game

This project was a simple text-scripted remake of a classic solitaire chess game. The game is played by ridding the board of marbles by jumping pieces into empty spaces. The game allows players to switch between three boards: English, European, and a triangle board. You can also customize the boards' size and where the empty slot should be placed through the command arguments.
